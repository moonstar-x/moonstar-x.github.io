# My site

This repo contains the code for my main [website](https://moonstar-x.dev), made in Svelte.

## Development

To develop this site, install the dependencies:

```text
npm install
```

And run the development server:

```text
npm run dev
```

## Building

To build the site, run:

```text
npm run build
```
